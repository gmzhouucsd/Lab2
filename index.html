New: 2.0 High Level Planning of Planner

  • Home page will be a calendar displaying the days in the month, and clicking one of the days zooms you into the day for daily planning
  • Will include UI to allow users to add events to days. This will include the name, description, and time period of the events
  • Once events are finished, they will be crossed out, so the user can easily understand what events are up next
  • If we have time, we can also add an in-app reward system for successfully completed events
